現在先頭のキャラ:
現在 行目
使い方
 コマンドは半角で、またスペース区切りで設定すること
[]内が引数(かっこ自体は書かない)
()内のものは省略できる(バフなど)

#[comment]
 #から始まるものは行末まで無視されます(コメントに使用)
命令の最後に書いてもよい
例:
move 125 enemy
#ここからデバフを入れる
例:
move 125 momo #とっておき

set [name] [SPD] ([buff(%)])
開始(start)前にフィールドに出すキャラを設定
SPDの優先度の高いキャラから先に書く
nameにスペースを含めるのは禁止

start
メインモード切り替え
setを書き終わったら記入

end
メインモードモード終了
その後ろにstart、start_sortを書くことでまた別のモードに切り替えられる

move [硬直(%)] [name]
行動
先頭のキャラ以外をnameに記入した場合errorを吐く
moveの代わりにmでも可

start_sort
ソートモード
end_sortを記入すると誰かのmove_listで書かれた硬直リストがなくなるまで行動する
(他のキャラにまだ硬直リストが残っていても終了)

end_sort
ソートモード終了
これを書かないとソートされない
その後ろにstart、start_sortを書くことでまた別のモードに切り替えられる

move_list [name] [[硬直(%)],[硬直(%)],...]
行動リスト
move_list nagi [40,100,100]
のように書く
スペースを空けるのは厳禁
move_listの代わりにmv_lsでも可


----ここからほとんどデバッグしていないので正しく動かない可能性大----

buffset [name] [buff(%)]
buffadd [name] [buff(%)]
buffminus [name] [buff(%)]
バフ設定
それぞれ b,b+,b-と省略可

switch [from_name] [to_name] [SPD] ([buff])
from_nameに交代前のキャラ名、to_nameに交代後のキャラ名を記入
swと省略可
